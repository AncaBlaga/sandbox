# Counting_words

This script will parse a file, count the times each word appears and will output the first 5 words most used and the number they appear in the file.

## Requirements
Python 3.12 or higher

## Installation
No installation needed

## Usage

To run the script:
```bash
python counting_words.py sample_text.txt
```

You can use any file as long as you provide the relative path to the file.

## Example Output
```
[('the', 20),
 ('fox', 8),
 ('dog', 8),
 ('quick', 5),
 ('was', 4)]
```
